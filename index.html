Moodify Me

Currently in development with the goal of serving as a web application that allows users to send messages to each other with expressive emojis attached at the end of the message. These emojis are carefully picked through a machine learning algorithm that quantifies each text message. Each text message will be categorized into a specific "mood" which corresponds to that emoji.
